# US Lead Pipeline Sankey
Deploy to GitHub Pages to embed in HubSpot.

## Setup
1. Create a new GitHub repo (e.g. `lead-sankey`)
2. Push this folder's contents to the `main` branch
3. Go to **Settings → Pages → Source: Deploy from branch → main / root**
4. Your URL will be `https://<org>.github.io/lead-sankey/`
5. In HubSpot, add an **Embed** card to your dashboard and paste the URL

## Updating data
Replace the `LEADS_RAW` array in index.html with fresh export data.
